# 🎨 Полный список ЭФФЕКТОВ (`type:`) и их параметров

## `PARTICLE` — Обычные частицы

| Параметр | Пример | Описание |
|---|---|---|
| `particle` | `FLAME` | Любое имя частицы из игры: `HEART`, `LAVA`, `SMOKE`, `SOUL_FIRE_FLAME`... |
| `amount` | `20` | Количество частиц |
| `speed` | `0.1` | Скорость разлета |
| `offset.x / y / z` | `0.2` | Радиус разброса зоны спавна |

---

## `GEOMETRY` — 3D фигуры из частиц

| Параметр | Пример | Описание |
|---|---|---|
| `shape` | `SPHERE` | Формы: `SPHERE`, `CUBE`, `TORNADO`, `STAR` |
| `particle` | `END_ROD` | Любая частица из игры |
| `size` | `1.5` | Размер или радиус фигуры |

---

## `FIREWORK` — Кастомные салюты

| Параметр | Пример | Описание |
|---|---|---|
| `shape` | `BALL` | Формы: `BALL`, `LARGE_BALL`, `STAR`, `BURST`, `CREEPER` |
| `colors` | `["FF0000", "00FF00"]` | Список цветов в формате HEX без решетки |
| `flicker` | `true` / `false` | Эффект мерцания звезд после взрыва |
| `trail` | `true` / `false` | Дымный след при взлете |
| `power` | `1` | Высота полета до взрыва, от 1 до 3 |

---

## `SOUND` — Звуки из игры

| Параметр | Пример | Описание |
|---|---|---|
| `sound` | `ENTITY_PLAYER_LEVELUP` | Любая константа звука Bukkit API |
| `volume` | `1.0` | Громкость |
| `pitch` | `1.0` | Тональность/скорость |

---

## `TITLE` — Текст на весь экран

| Параметр | Пример | Описание |
|---|---|---|
| `title` | `"<gradient:gold:yellow>Заголовок</gradient>"` | Поддерживает MiniMessage цвета |
| `subtitle` | `"<white>Подзаголовок</white>"` | — |
| `fadeIn` | `20` | Появление в тиках, 20 тиков = 1 сек |
| `stay` | `60` | Отображение на экране в тиках |
| `fadeOut` | `20` | Исчезновение в тиках |

---

## `EXPLOSION` — Взрывы

| Параметр | Пример | Описание |
|---|---|---|
| `explosion-type` | `NORMAL` | `NORMAL` — обычный взрыв, `HUGE` — мега-взрыв |
| `break-blocks` | `true` / `false` | Будет ли взрыв ломать блоки и наносить урон миру |
| `power` | `2.0` | Радиус и сила физического взрыва |

---

## `LIGHTNING` — Молнии

| Параметр | Пример | Описание |
|---|---|---|
| `visual-only` | `true` / `false` | `true` — чисто эффект без урона, `false` — настоящая молния, бьющая током и поджигающая |
