# Полный список ТРИГГЕРОВ (`trigger:`)

| Название триггера | Что указывать в `target:` (регистр важен!) | Описание события |
|---|---|---|
| `BLOCK_PLACE` | Название блока (`DIAMOND_BLOCK`, `STONE`) | Игрок поставил блок |
| `BLOCK_BREAK` | Название блока (`OAK_LOG`, `DIRT`) | Игрок сломал блок |
| `BLOCK_INTERACT` | Название блока (`CHEST`, `LEVER`) | Клик (ЛКМ/ПКМ) по блоку |
| `CRAFT_ITEM` | Название предмета (`CRAFTING_TABLE`) | Игрок скрафтил предмет |
| `EAT` | Название еды (`GOLDEN_APPLE`) | Игрок съел еду |
| `DRINK` | Название зелья/ведра (`POTION`, `MILK_BUCKET`) | Игрок выпил что-то |
| `DROP_ITEM` | Название предмета (`DIAMOND`) | Игрок выбросил предмет |
| `PICKUP_ITEM` | Название предмета (`IRON_INGOT`) | Игрок подобрал предмет |
| `OPEN_CHEST` | `CHEST` | Игрок открыл обычный сундук |
| `OPEN_INVENTORY` | Тип инвентаря (`ANVIL`, `FURNACE`) | Открытие других контейнеров |
| `DAMAGE` | Причина урона (`FALL`, `LAVA`, `FIRE`) | Игрок получил урон |
| `DEATH` | Не используется | Игрок умер |
| `KILL_PLAYER` | Никнейм игрока | Игрок убил другого игрока |
| `KILL_ENTITY` | Тип моба (`CREEPER`, `ZOMBIE`) | Игрок убил моба |
| `TELEPORT` | Причина (`ENDER_PEARL`, `COMMAND`) | Игрок телепортировался |
| `PORTAL` | Причина (`NETHER_PORTAL`, `ENDER_PORTAL`) | Игрок вошел в портал |
| `PLAYER_COMMAND` | Команда целиком (`/spawn`, `/home`) | Игрок ввел команду |
| `SEND_MESSAGE` | Текст сообщения | Игрок написал в чат |
| `WORLD` | Имя мира (`world_nether`, `world_the_end`) | Игрок сменил мир |
| `LEVEL_UP` | Число уровня (`30`) | Игрок поднял уровень опыта |
| `ADVANCEMENT` | ID ачивки (`minecraft:story/mine_diamond`) | Получено достижение |
| `BUCKET_FILL` | Название ведра (`WATER_BUCKET`) | Набрал ведро жидкости |
| `BUCKET_EMPTY` | Название ведра (`LAVA_BUCKET`) | Вылил ведро жидкости |
| `JOIN` / `QUIT` | Не используется | Вход / Выход с сервера |
| `RESPAWN` | Не используется | Возрождение на спавне/кровати |
| `WAKEUP` | Не используется | Игрок встал с кровати |
| `BED_ENTER` / `BED_LEAVE` | Не используется | Лег в кровать / Встал с нее |
| `MOVE` / `JUMP` | Не используется | Движение на блок / Прыжок |
| `SNEAK` / `SPRINT` | Не используется | Сел на корточки / Побежал |
| `SWIM` / `GLIDE` / `FLY` | Не используется | Плавание / Элитры / Fly-режим |
